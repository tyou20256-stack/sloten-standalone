スロット天国サポート — ウィジェットデモ

右下のチャットボタンをクリックすると、AI サポートチャットが開きます。

埋め込みコード:
<script src="https://<worker>/widget/widget.js" data-api="https://<worker>" async></script>
ユーザー情報を渡す場合 (推奨):
<script src=".../widget.js"
  data-api=".../"
  data-user-name="山田太郎"
  data-user-email="taro@example.com"
  data-user-external-id="sloten_user_12345"
  async></script>

または JavaScript で:
window.SlotenChatConfig = { user: { name: "...", email: "...", external_id: "...", metadata: { vip_rank: "gold" } } };
